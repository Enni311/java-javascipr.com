Что такое Java?

Java — это мощный объектно-ориентированный язык программирования, который используется для разработки разнообразных приложений: от мобильных приложений (Android) до крупных корпоративных систем.

Основные концепции Java:

  • Объекты и классы
  • Инкапсуляция
  • Наследование
  • Полиморфизм
  • Интерфейсы
  • Исключения
  • Модификаторы доступа

Что такое Класс в Java?

В Java, класс — это чертеж или шаблон для создания объектов. Он определяет структуру и поведение, которыми будут обладать все объекты этого класса. Класс описывает, какие данные (поля или атрибуты) будут хранить объекты, и какие действия (методы) они смогут выполнять.

Для чего нужен класс? Классы позволяют организовывать код, делая его более модульным, понятным и поддерживаемым. Они являются основой объектно-ориентированного программирования, позволяя создавать сложные системы из взаимодействующих объектов.

Что такое Объект в Java?

Объект — это конкретный экземпляр класса. Когда вы создаете объект класса, вы выделяете память для хранения данных, определенных в классе, и получаете возможность вызывать методы этого класса для работы с этими данными.

Пример: Если у вас есть класс Dog (Собака), то myDog (моя собака) породы "Лабрадор" и возрастом 3 года будет конкретным объектом класса Dog.

Модификатор доступа public

Ключевое слово public является модификатором доступа в Java. Модификаторы доступа определяют уровень видимости (доступности) классов, интерфейсов, переменных (полей), методов и конструкторов. public предоставляет наивысший уровень доступа. Это означает, что элемент, объявленный как public, доступен из любого места в программе:

  • Из любого класса в том же пакете.
  • Из любого класса в другом пакете (при необходимости импортирования пакета или через полное имя класса).
  • Из любого подкласса (даже если он находится в другом пакете).

Когда и где использовать public:

  1. Публичные классы и интерфейсы: Классы (шаблоны для объектов) и интерфейсы (контракты для классов), которые предназначены для использования другими частями программы, обычно объявляются как public. Это делает их доступными для создания объектов и реализации интерфейсов в других классах.
  2. Публичные методы: Методы (действия, которые могут выполнять объекты), которые вы хотите сделать доступными для вызова из других классов, объявляются как public. Например, метод bark() в классе Dog, если он public, может быть вызван из любого другого класса, имеющего объект Dog. Важно отметить, что метод main(String[] args), с которого начинается выполнение Java-программы, всегда должен быть public static void, чтобы быть доступным для виртуальной машины Java (JVM).
  3. Публичные переменные (поля): Переменные (данные, которые хранят объекты), объявленные как public, могут быть напрямую доступны и изменены из любого места программы. Однако это считается плохой практикой, так как нарушает принцип инкапсуляции (сокрытия внутреннего состояния объекта). Рекомендуется использовать приватные переменные и предоставлять контролируемый доступ к ним через публичные методы (геттеры и сеттеры).
  4. Публичные конструкторы: Конструкторы (специальные методы для создания новых объектов класса), объявленные как public, могут быть вызваны из любого класса для создания новых объектов данного типа. Если конструктор не является public, создание объектов может быть ограничено определенными частями программы.

Что такое JavaScript?

JavaScript — это динамичный язык программирования, который используется для добавления интерактивности веб-страницам. Он выполняется в браузере пользователя.

Основные концепции JavaScript:

  • Переменные (var, let, const)
  • Типы данных (примитивные и объектные)
  • Операторы
  • Управляющие структуры (if-else, циклы)
  • Функции
  • Объекты и массивы
  • DOM (Document Object Model)
  • События

Примеры кода JavaScript с подробными пояснениями:

Переменные и типы данных:


// Объявление переменной с помощью 'let' (современный стандарт)
let message = "Привет!";
console.log(message); // Выведет: Привет!

// Изменение значения переменной
message = "Новое сообщение";
console.log(message); // Выведет: Новое сообщение

// Объявление константы с помощью 'const' (значение нельзя переназначить)
const PI = 3.14159;
console.log(PI);

// Попытка переназначить константу вызовет ошибку:
// PI = 3.14; // TypeError: Assignment to constant variable.

// Основные типы данных:
let name = "Иван"; // Строка (string)
let age = 30;      // Число (number)
let isStudent = false; // Булево значение (boolean)
let undefVar;       // Undefined
let nullVar = null; // Null

console.log(typeof name);      // Выведет: string
console.log(typeof age);       // Выведет: number
console.log(typeof isStudent); // Выведет: boolean
console.log(typeof undefVar);  // Выведет: undefined
console.log(typeof nullVar);   // Выведет: object (особенность JavaScript)

Условные операторы (if-else):


let temperature = 15;

if (temperature > 20) {
    console.log("На улице тепло.");
} else if (temperature > 10) {
    console.log("На улице прохладно.");
} else {
    console.log("На улице холодно.");
}

let isAdmin = true;
let action = isAdmin ? "разрешено" : "запрещено"; // Тернарный оператор
console.log("Действие " + action); // Выведет: Действие разрешено

Циклы (for, while):


// Цикл for
for (let i = 0; i < 5; i++) {
    console.log("Итерация for: " + i);
}

// Цикл while
let count = 0;
while (count < 3) {
    console.log("Итерация while: " + count);
    count++;
}

// Цикл for...of для перебора массивов
const colors = ["red", "green", "blue"];
for (const color of colors) {
    console.log("Цвет: " + color);
}

Функции:


// Объявление функции
function greet(personName) {
    return "Привет, " + personName + "!";
}

// Вызов функции
let greeting = greet("Анна");
console.log(greeting); // Выведет: Привет, Анна!

// Анонимная функция (функциональное выражение)
const multiply = function(a, b) {
    return a * b;
};

let product = multiply(4, 7);
console.log("Произведение: " + product); // Выведет: Произведение: 28

// Стрелочная функция (краткая запись)
const add = (x, y) => x + y;
let sum = add(10, 5);
console.log("Сумма: " + sum); // Выведет: Сумма: 15

Объекты:


// Объявление объекта
const person = {
    firstName: "Петр",
    lastName: "Иванов",
    age: 35,
    city: "Москва",
    greet: function() {
        console.log("Меня зовут " + this.firstName + " " + this.lastName + ".");
    }
};

// Доступ к свойствам объекта
console.log("Имя: " + person.firstName);   // Выведет: Имя: Петр
console.log("Возраст: " + person["age"]); // Выведет: Возраст: 35

// Вызов метода объекта
person.greet(); // Выведет: Меня зовут Петр Иванов.

// Добавление нового свойства
person.occupation = "Инженер";
console.log("Профессия: " + person.occupation); // Выведет: Профессия: Инженер

Массивы:


// Объявление массива
const numbers = [10, 20, 30, 40, 50];
console.log("Массив чисел: " + numbers); // Выведет: Массив чисел: 10,20,30,40,50

// Доступ к элементам массива по индексу
console.log("Первый элемент: " + numbers[0]); // Выведет: Первый элемент: 10
console.log("Третий элемент: " + numbers[2]); // Выведет: Третий элемент: 30

// Длина массива
console.log("Длина массива: " + numbers.length); // Выведет: Длина массива: 5

// Добавление элемента в конец массива
numbers.push(60);
console.log("Массив после добавления: " + numbers); // Выведет: Массив после добавления: 10,20,30,40,50,60

// Удаление последнего элемента из массива
numbers.pop();
console.log("Массив после удаления: " + numbers); // Выведет: Массив после удаления: 10,20,30,40,50
