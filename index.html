Что такое Java?

Java — это мощный объектно-ориентированный язык программирования, который используется для разработки разнообразных приложений: от мобильных приложений (Android) до крупных корпоративных систем.

Основные концепции Java:

  • Объекты и классы
  • Инкапсуляция
  • Наследование
  • Полиморфизм
  • Интерфейсы
  • Исключения
  • Модификаторы доступа

Модификатор доступа public

Ключевое слово public является модификатором доступа в Java. Модификаторы доступа определяют уровень видимости (доступности) классов, интерфейсов, переменных (полей), методов и конструкторов. public предоставляет наивысший уровень доступа. Это означает, что элемент, объявленный как public, доступен из любого места в программе:

  • Из любого класса в том же пакете.
  • Из любого класса в другом пакете (при необходимости импортирования пакета или через полное имя класса).
  • Из любого подкласса (даже если он находится в другом пакете).

Когда и где использовать public:

  1. Публичные классы и интерфейсы: Классы, которые предназначены для использования другими классами, и интерфейсы обычно объявляются как public.
  2. Публичные методы: Методы, которые вы хотите сделать доступными для вызова из других классов, объявляются как public. Метод main(String[] args) всегда должен быть public static void.
  3. Публичные переменные (поля): Хотя это возможно, но считается плохой практикой. Рекомендуется использовать приватные переменные и предоставлять доступ через публичные геттеры и сеттеры.
  4. Публичные конструкторы: Конструкторы, объявленные как public, могут быть вызваны из любого класса для создания новых объектов.

Примеры использования public:

Публичный класс и метод:


// public class Calculator - Объявление публичного класса с именем "Calculator".
public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        int sum = calc.add(5, 3);
        System.out.println("Сумма: " + sum);
    }
}

В этом примере класс Calculator и его метод add объявлены как public.

Публичный интерфейс и его реализация:


// public interface Printable
public interface Printable {
    void print();
}

// public class Document implements Printable
public class Document implements Printable {
    String content;
    public Document(String content) {
        this.content = content;
    }
    @Override
    public void print() {
        System.out.println("Содержимое документа: " + content);
    }
    public static void main(String[] args) {
        Printable doc = new Document("Важный текст");
        doc.print();
    }
}

Интерфейс Printable и класс Document объявлены как public.

Публичный конструктор:


// public class Person
public class Person {
    String name;
    int age;
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    public void displayInfo() {
        System.out.println("Имя: " + name + ", Возраст: " + age);
    }
    public static void main(String[] args) {
        Person person = new Person("Алиса", 30);
        person.displayInfo();
    }
}

Конструктор класса Person объявлен как public.

Что такое JavaScript?

JavaScript — это динамичный язык программирования, который используется для добавления интерактивности веб-страницам. Он выполняется в браузере пользователя.

Основные концепции JavaScript:

  • Переменные (var, let, const)
  • Типы данных (примитивные и объектные)
  • Операторы
  • Управляющие структуры (if-else, циклы)
  • Функции
  • Объекты и массивы
  • DOM (Document Object Model)
  • События

Примеры кода JavaScript с подробными пояснениями:

Переменные и типы данных:


// let message = "Привет!"; - Объявление переменной с именем "message" с использованием ключевого слова 'let'.
// 'let' используется для объявления переменных, которые могут быть переназначены.
// Переменной присваивается строковое значение "Привет!".
let message = "Привет!";

// const count = 10; - Объявление константы с именем "count" с использованием ключевого слова 'const'.
// 'const' используется для объявления переменных, которые не могут быть переназначены после инициализации.
// Константе присваивается числовое значение 10.
const count = 10;

// let isActive = true; - Объявление переменной с именем "isActive" с использованием 'let'.
// Переменной присваивается булевое значение 'true'. Булевые значения могут быть 'true' или 'false'.
let isActive = true;

// console.log(message); - Вывод значения переменной 'message' в консоль браузера.
// 'console.log()' - это функция, которая используется для вывода информации в консоль разработчика.
console.log(message); // Вывод: Привет!

// console.log("Количество: " + count); - Вывод строки "Количество: " и значения переменной 'count' в консоль.
// Оператор '+' используется здесь для конкатенации (объединения) строки и числа (которое будет автоматически преобразовано в строку).
console.log("Количество: " + count); // Вывод: Количество: 10

// console.log("Активный: " + isActive); - Вывод строки "Активный: " и значения переменной 'isActive' в консоль.
console.log("Активный: " + isActive); // Вывод: Активный: true

В этом примере показано объявление переменных различных типов (строка, число, булево значение) с использованием let и const, а также вывод их значений в консоль.

Функции:


// function greet(name) { ... } - Объявление обычной функции с именем "greet".
// Функция принимает один аргумент с именем "name".
function greet(name) {
    // return "Привет, " + name + "!"; - Функция возвращает строку, которая представляет собой приветствие,
    // объединяя строку "Привет, " с переданным значением аргумента 'name' и восклицательным знаком.
    return "Привет, " + name + "!";
}

// let greeting = greet("Иван"); - Вызов функции "greet" с аргументом "Иван".
// Возвращаемое значение функции ("Привет, Иван!") присваивается переменной 'greeting'.
let greeting = greet("Иван");

// console.log(greeting); - Вывод значения переменной 'greeting' в консоль.
console.log(greeting); // Вывод: Привет, Иван!

// const multiply = (a, b) => { ... }; - Объявление стрелочной функции с именем "multiply".
// Стрелочные функции - это более краткий способ записи функций в JavaScript.
// '(a, b)' - это список аргументов, которые принимает функция.
// '=>' - это "стрелка", которая отделяет аргументы от тела функции.
const multiply = (a, b) => {
    // return a * b; - Тело функции: возвращает результат умножения аргументов 'a' и 'b'.
    return a * b;
};

// let result = multiply(5, 3); - Вызов стрелочной функции "multiply" с аргументами 5 и 3.
// Возвращаемое значение функции (15) присваивается переменной 'result'.
let result = multiply(5, 3);

// console.log("Результат умножения: " + result); - Вывод строки "Результат умножения: " и значения переменной 'result' в консоль.
console.log("Результат умножения: " + result); // Вывод: Результат умножения: 15

Здесь показано объявление и использование как обычной функции (greet), так и стрелочной функции (multiply) в JavaScript.

Использование условного оператора if-else:


// let age = 20; - Объявление переменной 'age' и присвоение ей числового значения 20.
let age = 20;

// if (age >= 18) { ... } else { ... } - Условный оператор 'if-else'.
// Условие в скобках '(age >= 18)' проверяет, является ли значение переменной 'age' больше или равно 18.
if (age >= 18) {
    // console.log("Совершеннолетний"); - Этот блок кода выполняется, если условие 'age >= 18' истинно (true).
    console.log("Совершеннолетний"); // Вывод: Совершеннолетний
} else {
    // console.log("Несовершеннолетний"); - Этот блок кода выполняется, если условие 'age >= 18' ложно (false).
    console.log("Несовершеннолетний");
}

Пример использования условного оператора if-else в JavaScript для выполнения различных действий в зависимости от условия.

Использование цикла for:


// for (let i = 1; i <= 5; i++) { ... } - Цикл 'for'.
// Первая часть 'let i = 1;' - инициализация счетчика цикла, переменной 'i' присваивается начальное значение 1.
// Вторая часть 'i <= 5;' - условие продолжения цикла. Цикл будет выполняться до тех пор, пока значение 'i' меньше или равно 5.
// Третья часть 'i++;' - инкремент счетчика. После каждой итерации цикла значение 'i' увеличивается на 1.
for (let i = 1; i <= 5; i++) {
    // console.log("Число: " + i); - На каждой итерации цикла выводится строка "Число: " и текущее значение переменной 'i' в консоль.
    console.log("Число: " + i);
    // Вывод:
    // Число: 1
    // Число: 2
    // Число: 3
    // Число: 4
    // Число: 5
}
